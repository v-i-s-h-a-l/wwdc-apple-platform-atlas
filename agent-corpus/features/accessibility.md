### Evidence-Backed Claims

- Generated subtitles and subtitle styles combine accessibility, media quality, localization, and playback presentation. Source: WWDC 2026 session 256.
- Apps should consider subtitle generation, styling, readability, and user preference behavior together. Source: WWDC 2026 session 256.
- Built-in text views provide platform behavior, while custom TextKit views expose storage, layout, and rendering control. Source: WWDC 2026 session 370.
- New rendering surface APIs help customize text view rendering; UITextView and NSTextView can be wrapped into SwiftUI. Source: WWDC 2026 session 370.
- Reading apps need granular line, word, and character navigation. Source: WWDC 2026 session 219.
- Standard UIKit and SwiftUI text views already provide much of this; custom text should adopt UITextInput fully. Source: WWDC 2026 session 219.
- Device Hub ships with Xcode 27 as a single control surface for devices and simulators. Source: WWDC 2026 session 260.
- It supports live display, touch, hardware controls, zoom, resize, keyboard capture, diagnostics, app containers, profiles, and provisioning. Source: WWDC 2026 session 260.
- CSS Grid Lanes supports adaptive masonry-like layouts for varied content shapes while retaining structured CSS layout control. Source: WWDC 2026 session 314.
- Flow Tolerance helps balance visual layout flexibility with accessibility-preserving source and reading order. Source: WWDC 2026 session 314.
- The Now Playing framework centralizes media playback presence, controls, and system integration. Source: WWDC 2026 session 312.
- Media apps should treat Now Playing as part of the user experience across lock screen, remote controls, and platform surfaces. Source: WWDC 2026 session 312.
